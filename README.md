# Laravel Blog

Add a blog to your Laravel Application 

### Installation 

`$ composer require brucebrophy/laravel-blog`

### Publish Blog Views
`$ php artisan blog:publish --ui={tailwind || bootstrap}`

### Testing

You can run the tests with:

```bash
vendor/bin/phpunit
```
